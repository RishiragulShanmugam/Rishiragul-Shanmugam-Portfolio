However, in some situations, certain aspects of a graphtheoretical system may be uncertain. Use of fuzzy-graphical methods in dealing with ambiguity and vague notions is very natural. Fuzzy-graph theory has a large number of applications in modeling various real-time systems where the level of information inherent in the system varies with different levels of precision. In this internship activities, we have presented a new framework to handle fuzzy information by combining fuzzy sets with graph structures. Bottom Navbar Website

HtmlCssJavaScript

Placing navigation at the bottom makes it easily accessible with thumbs. Bottom navigation bars help declutter the top navigation area. It Will reduce the Space of the Apps or Websites. Provides a clear and consistent navigation pattern.

Too many navigation items can lead to user confusion so try to avoid too many icons in a single page because Inconsistent design can make the app look unprofessional.Failing to provide feedback on active navigation items can cause user frustration. It's Just a Sample Project so i Used 7 Sections with 7 Icons. Bottom navigation bars are commonly used in mobile apps. It Can be used in web applications, especially those with a mobile-first design approach. 3-5 navigation items are ideal to avoid clutter. Icons help reduce language barriers and save space so I Used Icons only Mode. If Wanted We can Use text labels for icons to improve accessibility. To Know Where we are Highlight the active navigation item.

Tapping on an icon takes the user to the associated destination. The icon that the user is currently viewing is highlighted. Use a bottom navigation bar when an app has three to five top-level destinations. It's a well-known feature in Android applications, and is used in many apps like Instagram, Facebook, and Twitter.

You can customize the look and feel of the bottom navigation bar. For example, you can change the size of the icons, or make the selected item appear different from the unselected items. To implement a bottom navigation bar in an Android app, you can use Android's Material Design Library.
